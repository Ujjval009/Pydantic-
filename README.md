# Pydantic Data Validation Project

A collection of Python scripts demonstrating robust data validation, serialization, and model management using Pydantic. This project showcases best practices for data modeling, custom validation, and nested models in Python.

## Features

- 📦 Data validation with Pydantic models
- 🏷️ Field and model-level custom validators
- 🔗 Nested models and computed fields
- 🔄 Serialization and deserialization
- 🧪 Example scripts for real-world scenarios
- 🛡️ Type enforcement and error handling

## Tech Stack

- **Python 3.12+**: Core programming language
- **Pydantic**: Data validation and settings management using Python type annotations

## Installation

1. **Clone the repository:**
	```bash
	git clone https://github.com/Ujjval009/Pydantic-.git
	cd pydantic
	```

2. **Create and activate a virtual environment:**
	```bash
	python3 -m venv .venv
	source .venv/bin/activate
	```

## API Documentation

This project is script-based and does not expose HTTP API endpoints. Each script demonstrates a specific Pydantic feature:

- **DataValidation.py**: Field constraints and type validation
- **FieldValidator.py**: Custom field-level validation
- **ModelValidator.py**: Model-level validation for interdependent fields
- **Serialization.py**: Model serialization and dictionary conversion
- **NestedModels.py**: Nested Pydantic models
- **computed_Feild.py**: Computed fields using Pydantic
- **TypeValidation.py**: Type enforcement and validation
